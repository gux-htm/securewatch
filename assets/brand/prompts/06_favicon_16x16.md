# Prompt: Favicon — 16x16

**Use:** Browser tab favicon (favicon.ico / favicon.png), bookmark icon

---

Recreate the Oh-My-Guard! logo as an **ultra-minimal 16x16 favicon**.

At 16x16 pixels only the absolute core elements can exist:
- A shield silhouette — just the outline shape, filled dark navy
- A single white exclamation mark (!) that fills the shield — this is the only interior element
- The shield border in cyan, 1px thick
- Nothing else — no padlock, no circuit traces, no rings

The result should look like: a tiny dark shield with a bright white "!" inside it, with a thin cyan border. That's it.

Color palette:
- Shield fill: dark navy #0A0F1E
- Shield border: cyan #00D4FF (1px)
- Exclamation mark: white #FFFFFF

Style: pixel-perfect, maximum contrast, absolutely minimal. Must be recognizable as a shield+! at 16x16. No anti-aliasing artifacts.

Output: 16x16px PNG. Also provide as .ico format with 16x16 and 32x32 sizes embedded.
