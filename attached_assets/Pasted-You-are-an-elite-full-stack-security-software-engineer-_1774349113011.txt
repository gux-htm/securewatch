**Must-Have Features (implement ALL of them – exactly as described)**

1. **VPN Management (OpenVPN Integration)**
   - Admin creates multiple isolated networks (e.g., IT-Network, Management-Network, Finance-Network).
   - One-click generation of .ovpn files with embedded client certificates.
   - Distribute .ovpn via dashboard (download or email).
   - Policy: Employees can ONLY connect to the corporate network via the issued .ovpn file.

2. **Pre-Connection Zero-Trust Verification**
   - Before any device is allowed to join a network or communicate:
     - Client agent sends its Digital Certificate (OpenSSL), MAC address, and current IP.
     - Server verifies against registered device database.
     - All three must match exactly → otherwise connection is rejected and alarm raised.

3. **Centralized File & Resource Monitoring (Client Agent)**
   - Admin defines "protected resources" (specific directories or files on any endpoint).
   - Client agent (using watchdog + platform-specific hooks) monitors every create/edit/delete/rename/view event in real time.
   - For every event:
     - Compute SHA-256 hash of file (before & after change)
     - Apply digital signature of the logged-in user (using their private key stored in agent keystore)
     - Record: user digital signature, file hash, MAC, IP, timestamp, action type, privileges used
   - All events are streamed securely to central server and stored immutably in PostgreSQL.

4. **Privilege & Access Control Engine**
   - Granular ACLs per user / group / device:
     - File/Folder level: View | Edit | Delete | Rename | Full Control
   - Privileges can only be changed through AegisGuard dashboard (no local OS bypass).
   - Client agent enforces privileges in real time (block unauthorized actions + log attempt).
   - Support for user groups (e.g., "IT-Admins", "Finance-Users").

5. **Firewall Engine**
   - Full stateful firewall rules (allow/deny by IP, port, protocol, MAC, user certificate).
   - Rules can be applied at network level (server) or pushed to individual client agents.
   - Live rule editor in dashboard with one-click apply.

6. **IPS / IDS Engine**
   - Signature-based detection (pre-loaded common attack patterns + custom rules).
   - Anomaly-based detection (baseline learning of normal traffic/behavior per network).
   - IPS mode: actively drop/block malicious packets/connections.
   - Use Scapy + nfqueue (Linux) or WinDivert (Windows) for packet inspection and prevention.
   - Real-time alerts with severity levels.

7. **Audit & Forensics Layer (OpenSSL Powered)**
   - Immutable audit trail for every security event.
   - In case of breach/attack: system can instantly show that attacker credentials (IP + MAC + digital signature) do not match registered device → immediate isolation.
   - Tamper-proof logs (signed with server master key).

8. **Multi-Network & Multi-Tenant Support**
   - Unlimited virtual networks with complete isolation.
   - Role-based access for admins (Super Admin, Network Admin, Auditor, etc.).

9. **Real-Time Dashboard & Alerting**
   - Live map of all connected devices, VPN status, active threats.

   - Searchable audit logs with filters and export (PDF/CSV with digital signatures).
