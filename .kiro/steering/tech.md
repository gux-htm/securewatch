# Tech Stack

## Monorepo

- Package manager: pnpm (enforced — npm/yarn are blocked)
- Workspace tool: pnpm workspaces (`pnpm-workspace.yaml`)
- Node.js: 24
- TypeScript: ~5.9, `composite: true` on every package, `moduleResolution: bundler`, `target: es2022`
- Shared TS options in `tsconfig.base.json`; root `tsconfig.json` holds project references for all libs

## TypeScript / Node Packages

| Layer | Package | Key libs |
|---|---|---|
| Frontend (React SPA) | `@workspace/Oh-My-Guard!` | React 19, Vite 7, TailwindCSS 4, shadcn/ui (Radix UI), TanStack Query, Wouter, Recharts, Framer Motion, Zod, react-hook-form |
| API server | `@workspace/api-server` | Express 5, Pino, CORS, cookie-parser, esbuild (CJS bundle) |
| DB layer | `@workspace/db` | Drizzle ORM, `drizzle-zod`, PostgreSQL (`pg` Pool) |
| API contract | `@workspace/api-spec` | OpenAPI 3.1 (`openapi.yaml`), Orval codegen |
| Generated Zod schemas | `@workspace/api-zod` | Auto-generated from OpenAPI — do not edit manually |
| Generated React Query hooks | `@workspace/api-client-react` | Auto-generated from OpenAPI — do not edit manually |
| Utility scripts | `@workspace/scripts` | Individual `.ts` files in `src/`, run via pnpm filter |

## Python Stack (Oh-My-Guard! native backend — `Oh-My-Guard!/`)

- Python 3.12
- FastAPI + Uvicorn
- SQLAlchemy 2.x + Alembic (migrations)
- PostgreSQL 16
- Redis (cache + real-time pub/sub)
- OpenVPN + Easy-RSA (managed via subprocess)
- cryptography + pyOpenSSL (RSA-PSS signing, CA)
- watchdog + psutil (file/resource monitoring)
- Scapy + netfilterqueue (Linux IPS) / WinDivert (Windows)
- HTMX + TailwindCSS + Jinja2 (server-rendered dashboard)
- pytest + httpx + pytest-asyncio (testing)

## Common Commands

```bash
# Install all dependencies
pnpm install

# Typecheck entire monorepo (always run from root)
pnpm run typecheck

# Build everything
pnpm run build

# Run frontend dev server
pnpm --filter @workspace/Oh-My-Guard! run dev

# Run API server dev
pnpm --filter @workspace/api-server run dev

# Push DB schema (development)
pnpm --filter @workspace/db run push

# Regenerate API client + Zod schemas from openapi.yaml
pnpm --filter @workspace/api-spec run codegen

# Run a utility script
pnpm --filter @workspace/scripts run <script-name>
```

## Environment Variables

- `DATABASE_URL` — required by `@workspace/db` and the Python backend
- `PORT` — required by the Vite dev server and API server
- `BASE_PATH` — required by Vite (`vite.config.ts`)

## Key Conventions

- Never edit files under `lib/api-zod/src/generated/` or `lib/api-client-react/src/generated/` — they are auto-generated by Orval
- All API changes start in `lib/api-spec/openapi.yaml`, then run codegen
- Typecheck must always be run from the workspace root (`pnpm run typecheck`), not inside individual packages
- `emitDeclarationOnly` — `tsc` only emits `.d.ts`; bundling is handled by Vite / esbuild
- Python code must be fully type-hinted and pass mypy; never hard-code secrets
